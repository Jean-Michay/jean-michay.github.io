Empresa


Misión, Visión y Valores

Creamos servicios y productos digitales para nuestros clientes. Soluciones reales y brillantes en el campo de aplicaciones web y móviles. Cada proyecto es un pequeño segmento de nuestra vida, tomamos dediciones enfocadas a resultados.

🧭 Misión

Nuestra misión define el conjunto de cualidades que muestran nuestra forma de trabajar.
Permitir que las personas puedan crecer junto a su negocio a través de internet, con transición digital, tecnología y resiliencia, puedan transformar sus posibilidades, su calidad de vida, su entorno y la economía.

🔭 Visión

Aquí expresamos nuestra aspiración a largo plazo, contestamos a la pregunta. ¿Que cambio queremos hacer en el mundo?
Vamos a ser la empresa mas centrada en los negocios de nuestros clientes; para construir una empresa enfocada en el impacto positivo de los mismos hacia nuestro planeta.

Experiencias


Experiencia en html

HTML es un lenguaje de marcado que nos permite iindicar la estructura de nuestro documento mediante varias etiquetas.
Empecé viendo los conceptos de HTML (HyperText Markup Language, Lenguaje de Marcas de Hipertexto). Lo primero que hice con esto fue una clásico estructura con su respetivo título y cuerpo. Después comencé a utilizar más etiquetas HTML para poner imagenes, escribir párrafos, insertar saltos de líneas, etc...
Algunos ejemplo de etiquetas en html son:

h1/h1, img src=”” alt=””, ul li /li /ul,
p /p, span /span, div/div, hr/...

Experiencia con css

Css es el lenguaje que define las apariencias de una estructura escrita en html.
Con CSS le di estilos a esa estructura de mi página web. Al mismo tiempo conocí más etiquetas de CSS las cuales me permitieron dar colores al las letras, poner estilos de letras, márgenes, etc...

Algunos ejemplos de etiquetas:

color: ;, background: ;, font-family: ;, font-weight: ;, font-size: ;, margin: ;, border: ;, padding: ; …

Conclusiones


HTML es la base que toda persona debe aprender para incursionarse a la vida del desarrollo web debido que html contribuye a la estructura de la página web mediante la utilizaciôn de una serie de etiquetas que los navegadores interpretan y muestran a los usuarios en forma de información.
